# claude-tutors

A collection of Claude skills for self-learners. More than just a tutor — Claude will help you build real understanding and teach you how to think.
 
## What are skills?
 
Skills are instruction files that change how Claude behaves. Install one, and Claude automatically uses it whenever you study that subject — no extra setup per conversation.
 
## Available skills
 
| Skill | Description |
|-------|-------------|
| [`tutor`](./tutor/) | Teaches any subject for genuine, transferable understanding. Adapts to four situations: explaining a concept (clear and direct), working a problem with you (guides thinking, never hands over the answer), checking your work (diagnoses where reasoning breaks, not just right/wrong), and mapping a learning goal (finds your starting point and first concrete step). Grounded in established pedagogy — relational understanding, productive struggle, fighting the illusion of competence. |
| [`programming-tutor`](./programming-tutor/) | Mentors you in programming for real understanding — reasoning about code, debugging your own mistakes, designing solutions — not collecting snippets that run. Adapts to five situations: explaining a concept, writing code you're stuck on (guides, never hands over working code), debugging (teaches the hypothesis-test-narrow method, not just the fix), reviewing working code, and mapping a learning goal. Built on CS-education research: the notional machine, tracing before writing, and debugging as the scientific method. Language-agnostic. |
| [`math-tutor`](./math-tutor/) | Teaches math for genuine understanding, not memorization. Adapts to three situations: working a problem with you (guides thinking, never hands over steps), explaining a concept (clear and direct), or checking your solution (diagnoses where reasoning breaks, not just right/wrong). Covers algebra through calculus, probability, linear algebra, and beyond. |
 
## How to install

1. Download the skill folder as a ZIP:
   - Open the skill folder (e.g. [`math-tutor`](./math-tutor/))
   - Click the green **Code** button → **Download ZIP**
2. In [claude.ai](https://claude.ai): go to **Settings → Customize → Skills → +** → **Upload a skill**
3. Upload the ZIP file
4. That's it — Claude will use it automatically when you study
